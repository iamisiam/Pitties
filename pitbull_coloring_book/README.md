# Pitbull-Themed Coloring Book

A professional SVG coloring book featuring multiple poses and expressions of pitbulls.

## Structure
- **svg/**: Source SVG files
- **pages/simple/**: Easy difficulty (5-10 lines)
- **pages/medium/**: Medium difficulty (10-20 lines)
- **pages/complex/**: Complex difficulty (20+ lines)
- **output/**: Final printable PDFs/SVGs

## Page Size
- Standard 8.5" x 11" (US Letter)
- 0.5" margins on all sides
- 7.5" x 10" art area
